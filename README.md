# 旅伴分帳 App

多人共享的旅遊記帳分帳 App，支援多幣別、自動結算最少轉帳次數。

## 網址

- **App**：https://您的帳號.github.io/travel-split/
- **使用說明**：https://您的帳號.github.io/travel-split/guide.html

## 技術組成

- 前端：單一 HTML 檔案（純 JS + CSS，無框架）
- 後端：Firebase Firestore
- 部署：GitHub Pages

## 更新方式

1. 在 GitHub 網頁點 `index.html` → 右上角鉛筆圖示編輯
2. 或直接拖曳新檔案到 repo 覆蓋舊檔
3. 提交後約 1～2 分鐘自動部署完成

## 檔案結構

- `index.html` — App 主程式
- `guide.html` — 使用說明書
