# 旅伴分帳 App

多人共享的旅遊記帳分帳 App，支援多幣別、自動結算最少轉帳次數。

## 網址

- **App**：https://您的帳號.github.io/travel-split/
- **使用說明**：https://您的帳號.github.io/travel-split/guide.html

## 主要功能

- **即時同步**：所有旅伴透過 6 碼房間代碼共享同一份帳本，Firebase 即時推送
- **多幣別**：13 種常用幣別，可一鍵抓取 open.er-api.com 即時匯率
- **每筆自訂匯率**：同一幣別也能標記不同匯率（例如現金 vs 信用卡）
- **個人顯示幣別**：每位旅伴可獨立選擇要看哪種幣別，互不影響
- **多人付款**：一筆消費可由多人各付不同金額（例：Wayne 付 100、Lina 付 50）
- **靈活分攤**：平均分攤或每人指定金額，自動驗證合計
- **借款記錄**：純現金借款（A 借錢給 B）會列入結算但不計入旅程花費
- **備註欄位**：每筆消費可加自由文字備註
- **個人付款帳戶**：自己標記這筆是現金/信用卡/哪張卡付的，僅本人可見（存於本機）
- **最少轉帳結算**：自動計算誰該付誰多少，最小化交易次數
- **批次匯入**：把消費明細丟給 ChatGPT/Claude，回傳的 `TS|...` 行可直接貼回 App 一次匯入
- **手機桌面 App**：iOS / Android 都可加到主畫面當原生 App 用

## 技術組成

- 前端：單一 HTML 檔案（純 JS + CSS，無框架）
- 後端：Firebase Firestore
- 部署：GitHub Pages

## 更新方式

1. 在 GitHub 網頁點 `index.html` → 右上角鉛筆圖示編輯
2. 或直接拖曳新檔案到 repo 覆蓋舊檔
3. 提交後約 1～2 分鐘自動部署完成

## 檔案結構

- `index.html` — App 主程式
- `guide.html` — 使用說明書

## 批次匯入格式

每行一筆，共 8 欄以 `|` 分隔：

```
TS|說明|金額|幣別|類別|日期|付款方式|分攤方式
```

- 付款方式：`Wayne` 或 `paid:Wayne=100,Lina=50`（合計須等於金額）
- 分攤方式：`equal:Wayne,Lina` 或 `custom:Wayne=80,Lina=70`（合計須等於金額）

範例：

```
TS|聚餐|150|TWD|food|2026-04-26|paid:Wayne=100,Lina=50|custom:Wayne=80,Lina=70
```
